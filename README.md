# coreprotect-clickhouse

A fork of CoreProtect that adds support for using [ClickHouse](https://clickhouse.com/) as a database backend.

## Getting started
coreprotect-clickhouse supports importing data from existing MySQL/SQLite databases, see [Migrating to Clickhouse](https://github.com/EarthMC/coreprotect-clickhouse/blob/master/docs/migrating-to-clickhouse.md) for a guide.
